# 미션 - 개발자 비상근무

## ⚙️ 구현 순서

### 1단계 - 도메인 설계
* [x] 비상근무 도메인 설계
  * 월, 일, 요일, 근무자 이름
* [x] 비상근무표(비상근무 리스트-일급 컬렉션) 설계
* [x] 월별 비상근무표(DTO로 사용) 도메인 설계
  * 월, 일, 요일, 공휴일 여부, 근무자명
* [x] 공휴일 도메인(enum) 설계
* [x] 근무자 도메인 설계

### 2단계 - 비상근무를 배정할 달(월)과 요일 입력
- [x] 비상 근무를 배정할 월과 시작 요일 입력 요청 메시지 출력
- [x] [사용자] 달(월)과 시작 요일 입력
  - 입력 유효성검사(구분자)
  - 도메인 유효성검사
    - 달 : 1~12월 검증
    - 요일 : 월~일 검증
- [x] 달(월)별 마지막 날을 계산하는 로직(클래스) 작성
- [x] 실패시 다시 입력할 수 있는 로직 구현
